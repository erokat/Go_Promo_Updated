3. Нажмите Начать развертывание и обновите эту страницу, призы появятся.

Инструкция по настройке Supabase ⚡

База данных не настроена в файле конфигурации. Сейчас приложение использует демо-данные ( Mock ).

Чтобы сайт начал сохранять данные в аккаунт Supabase, выполните следующие действия:

  1. Создайте бесплатный проект в панели управления Supabase.
  2. Перейдите в раздел SQL Editor своего проекта, откройте новый запрос, скопируйте содержимое файла supabase-schema.sql в вашем репозитории и нажмите кнопку Run. Это создаст все необходимые таблицы, триггеры и политики безопасности.
  3. Перейдите в настройки проекта: Project Settings -> API.
  4. Скопируйте полученные значения в файл config.json:
    • supabaseUrl — скопируйте значение из Project URL
    • supabaseAnonKey — скопируйте значение из Project API keys (anon public)
  5. Чтобы зарегистрировать администратора, перейдите в раздел Authentication -> Users -> Add user, выберите Create user, введите Email (например, admin@go-promo.local) и пароль. Нажмите кнопку создания — триггер СУБД автоматически создаст профиль в таблице администраторов!

    ⚠️ Если пользователь уже был создан ранее без триггера: либо удалите его и создайте заново, либо добавьте запись вручную через вкладку SQL Editor в Supabase:
    INSERT INTO public.admin_users (id, username) VALUES ('UID_ПОЛЬЗОВАТЕЛЯ_ИЗ_AUTH', 'admin');
